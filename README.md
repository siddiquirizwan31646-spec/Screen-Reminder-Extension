# ⏰ Screen Reminder Chrome Extension

A simple and powerful Chrome extension that helps you stay productive by setting smart reminders directly in your browser.

---

## 🚀 Features

- 🔔 Set one-time or repeating reminders  
- 🖥️ On-screen overlay alerts (even while browsing)  
- 🔕 Desktop notifications with persistent alerts  
- 🔁 Repeat reminders (15 min, 30 min, hourly, daily)  
- 🧠 Smart scheduling using Chrome alarms  
- 💾 Local storage (no data tracking)

---

## 📸 Preview

Clean, dark-themed UI with smooth animations and real-time reminder tracking.

---

## 🛠️ Tech Stack

- JavaScript (Vanilla)
- Chrome Extension API (Manifest V3)
- HTML + CSS
